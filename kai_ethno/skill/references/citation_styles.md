# Estilos de Citación Académica

## APA 7th Edition (American Psychological Association)

### Libro
**Formato**: Apellido, A. A. (Año). *Título del libro*. Editorial.

**Ejemplo**:
```
Geertz, C. (1973). *The interpretation of cultures: Selected essays*. Basic Books.
```

### Capítulo de libro
**Formato**: Apellido, A. A. (Año). Título del capítulo. En A. Editor & B. Editor (Eds.), *Título del libro* (pp. xx-xx). Editorial.

**Ejemplo**:
```
Hine, C. (2015). Ethnography for the internet: Embedded, embodied and everyday. En M. V. 
Clancy & J. M. Corti (Eds.), *Digital anthropology* (pp. 45-62). Routledge.
```

### Artículo de revista
**Formato**: Apellido, A. A. (Año). Título del artículo. *Nombre de la Revista*, *Volumen*(Número), pp-pp. https://doi.org/xxxx

**Ejemplo**:
```
Pink, S., Horst, H., Postill, J., Hjorth, L., Lewis, T., & Tacchi, J. (2016). Digital 
ethnography: Principles and practice. *Qualitative Research*, 16(1), 108-124. 
https://doi.org/10.1177/1468794115611560
```

### Fuente en línea
**Formato**: Apellido, A. A. (Año). Título de la página. *Nombre del Sitio*. URL

**Ejemplo**:
```
Boellstorff, T. (2020). An interview with Tom Boellstorff on digital anthropology. 
*Cultural Anthropology*. https://culanth.org/fieldsights/an-interview-with-tom-boellstorff
```

## Chicago Manual of Style 17th Edition

### Notas al pie (Humanidades)

**Primera cita**:
```
1. Clifford Geertz, The Interpretation of Cultures: Selected Essays (New York: Basic Books, 1973), 5.
```

**Citas subsecuentes**:
```
2. Geertz, Interpretation of Cultures, 9.
```

### Bibliografía

**Libro**:
```
Geertz, Clifford. 1973. The Interpretation of Cultures: Selected Essays. New York: Basic Books.
```

**Artículo**:
```
Pink, Sarah, Heather Horst, John Postill, Larissa Hjorth, Tania Lewis, and Jo Tacchi. 2016. 
"Digital Ethnography: Principles and Practice." Qualitative Research 16 (1): 108-124.
```

## MLA 9th Edition (Modern Language Association)

### Obra en colección
**Formato**: Apellido, Nombre. "Título del ensayo." *Título de la colección*, editado por Nombre Editor, Editorial, Año, pp. xx-xx.

**Ejemplo**:
```
Hine, Christine. "Ethnography for the Internet: Embedded, Embodied and Everyday." 
*Digital Anthropology*, edited by Haidy Geismar and Hannah Knox, Routledge, 2020, pp. 45-62.
```

### Artículo de revista
**Formato**: Apellido, Nombre. "Título del artículo." *Título de la Revista*, vol. X, no. X, Año, pp. xx-xx.

**Ejemplo**:
```
Pink, Sarah, et al. "Digital Ethnography: Principles and Practice." *Qualitative Research*, 
vol. 16, no. 1, 2016, pp. 108-124.
```

## Estilos Específicos de Disciplina

### AAA (American Anthropological Association)
Basado en Chicago, adaptado para antropología.

**Libro**:
```
Geertz, Clifford. 1973. *The Interpretation of Cultures: Selected Essays*. New York: Basic Books.
```

**Artículo**:
```
Pink, Sarah, Heather Horst, John Postill, Larissa Hjorth, Tania Lewis, and Jo Tacchi. 2016. 
Digital Ethnography: Principles and Practice. *Qualitative Research* 16(1):108-124.
```

## Citas de Fuentes Digitales

### Plataformas de Redes Sociales
```
@username. (Fecha). Texto del post [Tipo de publicación]. Plataforma. URL
```

**Ejemplo Twitter**:
```
@digitalethno. (2023, 15 de marzo). New article on digital methods in anthropology 
[Tweet]. Twitter. https://twitter.com/digitalethno/status/1234567890
```

**Ejemplo Instagram**:
```
@research_visuals. (2023, 20 de marzo). Fieldwork photos from [lugar] [Fotografía]. 
Instagram. https://www.instagram.com/p/ABC123/
```

### Sitios Web
```
Autor/Autores. (Año). Título de la página. Sitio Web. URL. (Fecha de acceso)
```

### Preprints y Repositorios
```
Apellido, A. A. (Año). Título del preprint. Repositorio. URL. Versión X
```

## Herramientas de Gestión de Referencias

### BibTeX
```bibtex
@book{geertz1973interpretation,
  title={The Interpretation of cultures},
  author={Geertz, Clifford},
  year={1973},
  publisher={Basic Books}
}

@article{pink2016digital,
  title={Digital ethnography: Principles and practice},
  author={Pink, Sarah and Horst, Heather and Postill, John and Hjorth, Larissa and Lewis, Tania and Tacchi, Jo},
  journal={Qualitative Research},
  volume={16},
  number={1},
  pages={108--124},
  year={2016},
  publisher={SAGE Publications}
}
```

## Plantillas de Citas KAI-Ethno

### Cita de entrevista etnográfica
```
Entrevistado, N. (Fecha). Entrevista realizada por Investigador. Lugar. 
[Descripción: entrevista semiestructurada, 45 minutos].
```

### Cita de observación de campo
```
Investigador. (Fecha). Notas de campo. [Lugar de observación]. 
[Duración y tipo de observación].
```

### Cita de material archivado
```
Nombre del archivo. (Fecha). [Tipo de material]. Colección. 
Archivo/Repositorio. URL (si aplica).
```

## Errores Comunes

1. **Iniciales vs. nombres completos**: En APA usar solo iniciales de nombres
2. **Mayúsculas en títulos**: Solo primera palabra y nombres propios en APA/MLA
3. **Itálicas**: Solo títulos de libros y revistas, no artículos
4. **Fechas de acceso**: Obligatorias para fuentes digitales inestables
5. **DOI vs URL**: Preferir DOI cuando esté disponible
